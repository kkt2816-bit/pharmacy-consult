============================================================
  약국 상담 정리 프로그램 — 설치 & 사용 안내 (윈도우용)
============================================================

상담 전사문을 붙여넣으면 →
  ① AI가 상담 내용을 정리하고
  ② 구글시트에 한 줄로 저장하고 (상세 상담기록 포함)
  ③ 손님께 드릴 A4 안내문을 만들어 PDF로 뽑을 수 있어요.


────────────────────────────────────────────
■ 처음 한 번만 하면 되는 준비 (3가지)
────────────────────────────────────────────

[1] Node.js 설치
   - https://nodejs.org 접속 → 'LTS' 버튼 눌러 설치
   - 설치 중 옵션은 전부 '다음'으로 넘어가면 됩니다.

[2] 프로그램 실행
   - 이 폴더 안의  start.bat  파일을 더블클릭
   - 검은 창이 뜨고, 잠시 후 브라우저가 자동으로 열립니다.
     (안 열리면 브라우저 주소창에  http://127.0.0.1:8800  직접 입력)
   - ※ 검은 창은 프로그램이 켜져 있는 동안 닫으면 안 됩니다.
     끌 때는 검은 창을 닫으면 프로그램이 종료됩니다.

[3] 화면 오른쪽 아래 '⚙️ 설정' 열고 아래 3개 입력
   (1) AI 키        : 남편(형) API 키를 한 번만 붙여넣기  (sk-ant-... )
   (2) 구글시트 연결 : 아래 '구글시트 준비' 참고
   (3) 시트 URL/탭  : 저장할 구글시트 주소와 탭 이름
   입력 후 [설정 저장] → [구글시트 로그인] (아내분 구글 계정으로)

   ※ (선택·강력추천) 'OpenAI 백업 키' 칸에 OpenAI 키(sk-...)도 넣어두면,
     Claude(Anthropic) 서버가 과부하/장애일 때 자동으로 최신 OpenAI(gpt-5.5)로
     넘어가서 AI가 멈추는 일이 거의 없어집니다.


────────────────────────────────────────────
■ 구글시트 준비 (구글 클라이언트 ID / 시크릿)
────────────────────────────────────────────
※ 남편이 쓰던 OAuth 클라이언트를 그대로 재사용하면 됩니다.
   (새로 만들 필요 없음. 단, 아래 리디렉션 주소만 추가)

1. https://console.cloud.google.com 접속 (남편 계정)
2. 'API 및 서비스' → '사용자 인증 정보(Credentials)'
3. 쓰던 OAuth 2.0 클라이언트 클릭
4. '승인된 리디렉션 URI'에 아래 주소를 추가하고 저장:

      http://127.0.0.1:8800/auth/sheets/callback

   (설정 화면에도 이 주소가 표시되니 복사해서 붙여넣으면 됩니다)

5. 클라이언트 ID / 시크릿을 설정 화면에 입력
6. 저장할 구글시트를 미리 하나 만들어 두고, 그 주소(URL)를 입력
   - 첫 줄(머리글)은 프로그램이 자동으로 채워주니 비워도 됩니다.
   - '아내분' 구글 계정으로 [구글시트 로그인] 하면 그 계정 시트에 저장됩니다.


────────────────────────────────────────────
■ 매일 사용하는 법
────────────────────────────────────────────
1. start.bat 더블클릭 → 브라우저 열림
2. 왼쪽에 상담 전사문 붙여넣기 → [🧠 AI 정리]
3. 정리된 고객명·전화번호·상담일 등 확인/수정
4. [💾 구글시트 저장]  → 시트에 한 줄 추가됨
5. [📄 안내문 만들기]  → 오른쪽에 안내문이 뜸
6. 오른쪽 내용 직접 수정 가능 → [📄 PDF 생성]
   - 인쇄창이 뜨면 '대상'을 'PDF로 저장'으로 선택해 저장
   - (머리글/바닥글은 자동으로 빠지게 되어 있어요)

※ 안부문자 연락 알림은 '상담일 기준 약 3주 뒤(약 떨어지기 전)'에
   자동으로 뜹니다. 상담일만 정확히 입력하면 됩니다.

※ 설정 4단계를 다 끝내면 화면 오른쪽이 '처음 설정 가이드' 대신
   '📖 사용법'으로 바뀝니다. 쓰다가 막히면 거기를 보면 됩니다.


────────────────────────────────────────────
■ 안부·상담 문자 (위쪽 '💬 안부문자' 탭)
────────────────────────────────────────────
1. 위쪽 [💬 안부문자] 탭을 누르면, '상담일 기준 약 3주 뒤
   (약 떨어지기 전)' 연락할 분이 오늘 / 1일 뒤 / 2일 뒤로
   나뉘어 보입니다. (이름·전화번호만 표시)
2. 보낼 문자 내용은 상담을 저장할 때 미리 만들어져서,
   구글시트 맨 끝 칸('안부 상담문자' = S열)에 자동으로 적힙니다.
3. 그래서 연락할 분을 확인하고 → 구글시트에서 그 분 줄의
   맨 끝 칸 문자를 복사해 보내면 됩니다.


────────────────────────────────────────────
■ 자주 묻는 것
────────────────────────────────────────────
Q. 검은 창을 실수로 닫았어요.
 → 프로그램이 꺼진 것뿐입니다. start.bat 다시 더블클릭하면 됩니다.

Q. 'AI 키 필요' 라고 떠요.
 → 설정에서 AI 키를 저장하지 않았거나 잘못 입력된 경우입니다.

Q. 저장이 안 돼요 / '로그인' 관련 메시지가 떠요.
 → 설정에서 [구글시트 로그인]을 다시 한 번 해주세요.

Q. 입력한 키/시크릿은 어디 저장되나요?
 → 이 폴더 안에만 저장됩니다(.ai-key.json 등).
    인터넷에 올라가지 않으니 이 폴더를 남에게 공유하지 마세요.
