🩺 약국 상담 정리 — 시작하기

아래 순서대로 딱 한 번만 따라 하면 됩니다. (5분이면 끝나요)

설치 (처음 한 번만)

1이 폴더를 압축 풀기

받은 약국 상담 프로그램.zip 파일을 마우스 오른쪽 클릭 → '압축 풀기 / 전체 압축 풀기'를 누르세요.

바탕화면이나 '문서' 폴더 등 찾기 쉬운 곳에 풀면 좋아요.

2Node.js 설치

프로그램을 돌리는 데 꼭 필요한 무료 프로그램이에요. 아래 버튼을 눌러 'LTS' 버전을 받고, 설치 중엔 계속 '다음(Next)'만 누르면 됩니다.

▶ Node.js 다운로드 (nodejs.org)

※ 이미 설치돼 있으면 이 단계는 건너뛰어도 됩니다.

3프로그램 켜기

압축 푼 폴더 안의 start.bat 파일을 더블클릭하세요.

  • 검은 창이 뜨고, 잠시 후 브라우저가 자동으로 열립니다.
  • 안 열리면 브라우저 주소창에 http://127.0.0.1:8800 직접 입력.
  • ⚠️ 검은 창은 프로그램이 켜져 있는 동안 닫지 마세요. (끌 땐 검은 창을 닫으면 종료)

※ '알 수 없는 게시자' 같은 보안 경고가 뜨면 '추가 정보 → 실행'을 누르면 됩니다.

세팅 (프로그램 화면 안에서)

4화면 안 '🚀 처음 설정 가이드' 따라 하기

프로그램이 열리면 오른쪽에 설정 가이드 4단계가 보여요. 화면 설명대로:

  • 1단계 — 남편(형) AI 키 한 번 붙여넣기
  • 2단계 — 구글 클라우드에 주소 1줄 추가 + 클라이언트 ID/시크릿 입력 (화면의 '복사' 버튼으로 주소를 그대로 붙여넣으면 됨)
  • 3단계 — 저장할 구글시트 주소 입력 → '1~3단계 저장'
  • 4단계 — '구글시트 로그인' → 와이프 구글 계정으로 로그인

끝난 단계는 ✅ 로 바뀌고, 4개가 다 ✅ 되면 준비 끝!

매일 쓰는 법
① start.bat 더블클릭 → ② 왼쪽에 상담 전사문 붙여넣기 → 🧠 AI 정리 → ③ 내용 확인/수정 → 💾 구글시트 저장 → ④ 📄 안내문 만들기 → 수정 후 📄 PDF 생성.

문제가 생기면: 검은 창을 실수로 닫았다면 그냥 start.bat를 다시 더블클릭하면 됩니다. 자세한 내용은 같은 폴더의 README.txt를 참고하세요.
